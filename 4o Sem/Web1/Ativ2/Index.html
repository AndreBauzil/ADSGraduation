 Título (ou Heading) que possui tamanhos entre 1 a 6.
<p>

Paragraph

 É o elemento de texto mais básico, com formatação simples de parágrafo, bastante utilizado.
<img> É utilizado para incorporar imagens diretamente na página web.
<table>
Exemplo de Tabela
Exemplo de Linha
Utilizado para criar uma tabela para organizar dados em filas e colunas. As tabelas consistem em linhas (definidas por elementos <tr>) e células (definidas por elementos <td> ou <th> para cabeçalhos). A tag <thead> é usada para definir o cabeçalho da tabela, <tbody> para o corpo e <tfoot> para o rodapé.
<nav>
Link Link2 Link3
É uma seção da página que aponta para outras páginas ou para outras áreas da página, ou seja, uma seção com links de navegação.
<div>
Isto é uma div
 É um container genérico para fluxo de conteúdo, que não representa nada. Pode ser utilizado para agrupar elementos para estilizar, ou porque eles compartilham valores de atributos.
